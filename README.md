### Comandos Git
#### Clonar repositorio existente
git clone URL
### Verificar o status do projeto/repositório
git status
### add novos arquivos ao repositorio
git add -A * //-A para pegar todos
### Comitar projeto
git commit -m "Mensagem"
### atualizar git nuvem
git push origin main
